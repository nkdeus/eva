Gradient Types

Four main gradient types available in EVA CSS

grad-linear Linear gradients
grad-radial Radial gradients
TEXT grad-linear-text
TEXT grad-radial-text

Direction Controls

Control gradient direction with intuitive classes

Main Directions
d-t to top ↑
d-r to right →
d-b to bottom ↓
d-l to left ←
Diagonal Directions
d-tl top-left ↖
d-tr top-right ↗
d-bl bottom-left ↙
d-br bottom-right ↘

Angle Controls

Precise angle control from 0° to 360° in 5° intervals

a-0 0 degrees
a-45 45 degrees
a-90 90 degrees
a-135 135 degrees

Note: Angles override directions. Don't use d-* and a-* together.

Example: grad-linear from-brand to-accent a-45

Text Gradients

Apply gradients directly to text for stunning typography effects

LINEAR grad-linear-text
RADIAL grad-radial-text
ANGLE with direction
FADE with angle

Perfect for: Titles, headers, call-to-action text, and branding elements

Example: grad-linear-text from-brand to-accent d-tr
