Real-time CSS Variables

The entire EVA theme is based on these CSS variables and HUE values:

Brand HUE 169° --brand-hue
Accent HUE 10° --accent-hue
Extra HUE 200° --extra-hue

Magic EVA: These 3 values automatically generate all the variations of colors you see on this page!

Each color (brand, accent, extra) uses custom lightness and chroma values, with only the HUE changing to create variations.
