Colors

The next generation of theming

Documentation

Ultra-modern color system based on OKLCH with automatic variations

Main colors: brand, accent, extra, dark, light

Fade variations: _ (65%), __ (35%), ___ (5%)

Brightness variations: -d (darker), -b (brighter), -d_ (darker_), -b_ (brighter_)

Properties: _c- (color), _bg- (background), _bc- (border), _f- (fill), _s- (stroke)

Real-time CSS Variables

The entire EVA theme is based on these CSS variables and HUE values:

Brand HUE 169° --brand-hue
Accent HUE 10° --accent-hue
Extra HUE 200° --extra-hue

Magic EVA: These 3 values automatically generate all the variations of colors you see on this page!

Each color (brand, accent, extra) uses custom lightness and chroma values, with only the HUE changing to create variations.

Gradients

Unlimited gradients variations

Documentation

Ultra-modern gradient system with complete control via Emmet-style classes

Types: grad-linear, grad-radial, grad-linear-text, grad-radial-text

Colors: from-[color] to-[color] (all color variations available)

Effects: animated, hover-shift, hover-scale, bg-size variants

Direction Controls

Main directions:

d-t (top) d-r (right) d-b (bottom) d-l (left)

Diagonal directions:

d-tl (top-left) d-tr (top-right) d-bl (bottom-left) d-br (bottom-right)

Example: grad-linear from-brand to-accent d-tr

Angle Controls

Custom angles: from a-0 to a-360 in 5° intervals

a-0 (0°) a-45 (45°) a-90 (90°) a-135 (135°) a-180 (180°) a-270 (270°)

Example: grad-linear from-brand to-accent a-45

Warning: Angles replace directions (don't use d-* and a-* together)

Text Gradients

Special types:

grad-linear-text - Linear gradient on text grad-radial-text - Radial gradient on text

How it works: Uses the same modifiers (from-*, to-*, d-*, a-*)

EXAMPLE

grad-linear-text from-brand to-accent

RADIAL

grad-radial-text from-extra to-light

Perfect for titles and catchy texts
